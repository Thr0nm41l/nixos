# Keybindings

Modifier key: `Super` (Windows key)

> AZERTY layout — workspace keys use the unshifted number row.

## Core

| Keybind | Action |
|---|---|
| `Super + Enter` | Open terminal (kitty) |
| `Super + Space` | App launcher (rofi drun) |
| `Super + Tab` | Quick links |
| `Super + L` | Lock screen (hyprlock) |
| `Super + Shift + F` | Toggle floating |
| `Super + Q` | Close window (or close quickshell panel if open) |

## Focus

| Keybind | Action |
|---|---|
| `Super + ←` | Move focus left |
| `Super + →` | Move focus right |
| `Super + ↑` | Move focus up |
| `Super + ↓` | Move focus down |

## Move window

| Keybind | Action |
|---|---|
| `Super + Ctrl + ←` | Move window left |
| `Super + Ctrl + →` | Move window right |
| `Super + Ctrl + ↑` | Move window up |
| `Super + Ctrl + ↓` | Move window down |

## Resize

| Keybind | Action |
|---|---|
| `Super + Shift + ←` | Shrink window horizontally |
| `Super + Shift + →` | Grow window horizontally |
| `Super + Shift + ↑` | Shrink window vertically |
| `Super + Shift + ↓` | Grow window vertically |

## Workspaces (AZERTY)

| Keybind | Physical key | Action |
|---|---|---|
| `Super + &` | `&` (1 key) | Switch to workspace 1 |
| `Super + é` | `é` (2 key) | Switch to workspace 2 |
| `Super + "` | `"` (3 key) | Switch to workspace 3 |
| `Super + '` | `'` (4 key) | Switch to workspace 4 |
| `Super + (` | `(` (5 key) | Switch to workspace 5 |
| `Super + -` | `-` (6 key) | Switch to workspace 6 |
| `Super + è` | `è` (7 key) | Switch to workspace 7 |
| `Super + _` | `_` (8 key) | Switch to workspace 8 |
| `Super + ç` | `ç` (9 key) | Switch to workspace 9 |
| `Super + à` | `à` (0 key) | Switch to workspace 10 |
| `Super + Shift + <key>` | same keys | Move window to that workspace |
