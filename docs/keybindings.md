# Keybindings

Modifier key: `Super` (Windows key)

## Core

| Keybind | Action |
|---|---|
| `Super + Enter` | Open terminal (kitty) |
| `Super + R` | Open app launcher (rofi) |
| `Super + B` | Open browser (firefox) |
| `Super + E` | Open file manager (thunar) |
| `Super + Q` | Close focused window |
| `Super + F` | Toggle fullscreen |
| `Super + Shift + Space` | Toggle floating |
| `Super + P` | Toggle pseudo-tiling |
| `Super + L` | Lock screen (hyprlock) |

## Focus

| Keybind | Action |
|---|---|
| `Super + ←` | Move focus left |
| `Super + →` | Move focus right |
| `Super + ↑` | Move focus up |
| `Super + ↓` | Move focus down |

## Resize

| Keybind | Action |
|---|---|
| `Super + Shift + ←` | Shrink window horizontally |
| `Super + Shift + →` | Grow window horizontally |
| `Super + Shift + ↑` | Shrink window vertically |
| `Super + Shift + ↓` | Grow window vertically |

## Workspaces

| Keybind | Action |
|---|---|
| `Super + 1-4` | Switch to workspace 1-4 |
| `Super + Shift + 1-4` | Move window to workspace 1-4 |

## Media

| Keybind | Action |
|---|---|
| `XF86AudioRaiseVolume` | Volume up |
| `XF86AudioLowerVolume` | Volume down |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioPlay` | Play/Pause |
| `XF86AudioNext` | Next track |
| `XF86AudioPrev` | Previous track |

## Screenshot

| Keybind | Action |
|---|---|
| `Print` | Screenshot full screen |
| `Super + Print` | Screenshot region selection |

## Mouse

| Keybind | Action |
|---|---|
| `Super + LMB drag` | Move window |
| `Super + RMB drag` | Resize window |
